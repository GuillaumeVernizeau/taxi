# T-AIA-902-BDX_4

## Install
1. Set up a Python virtual environment:

    ```bash
    python3 -m venv .venv
    source .venv/bin/activate  # On Windows use `venv\Scripts\activate`
    ```

2. Install the required dependencies:

    ```bash
    pip install -r requirements.txt
    ```

## Usage
```shell
./main
```
